Audio Classifier

This is a simple audio classifier for 20 different classes

Click Learn more to get more information on the model or upload your audio/video file (*.wav, *mp3, *.mp4) to classify it. (Note: There is a file duration limit to 60 seconds)

These are the current supported classes:

  • airplane
  • breathing
  • brushing teeth
  • can opening
  • car horn
  • cat
  • chainsaw
  • chirping birds
  • church bells
  • clapping
  • clock alarm
  • clock tick
  • coughing
  • cow
  • crackling fire
  • crickets
  • crow
  • crying baby
  • dog
  • door wood creaks
